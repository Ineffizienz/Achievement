Changelog

Hier werd ich versuchen euch immer auf dem neusten Stand zu halten, was Updates für die Oberfläche betrifft. Ich werd die jetzt kontinuierlich weiterentwickeln und auch versuchen auf eure Wünsche einzugehen, sofern ich technisch dazu in der Lage bin ;)

Version 0.1.6

Features:
  • Möglichkeit die Sichtbarkeit der Achievements zu bestimmen, hinzugefügt
  • Bilder können jetzt nachträglich hochgeladen und verändert werden
Changes:
  • Designänderungen
Bugfixes:
  • Bug beim Erstellen der Achievements behoben
  • Bug in der Datenvalidierung behoben

Version 0.1.5

Features:
Changes:
  • Die maximale Zeichenanzahl bei Kategorie-, Trigger- und Achievementtiteln wurde auf 20 reduziert
  • Security Update
Bugfixes:
  • Fehler behoben, bei dem im Overview jedes Achievement unter jeder Kategorie angezeigt wurde
  • Grafikfehler bei den Achievements behoben

Version 0.1.4

Features:
  • Löschen-Funktion für Kategorien hinzugefügt
  • Löschen-Funktion für Trigger hinzugefügt
  • Platzhalter für Achievements ohne Bilder eingefügt
  • Achievements können jetzt nachträglich einer Kategorie und/oder einem Trigger zugewiesen werden
Changes:
  • der Overview ist wieder aktiv und überarbeitet
  • die maximal zulässige Anzahl an Zeichen beim Erstellen von Achievements wurde auf 150 reduziert
  • zu große Dateien werden jetzt mit einer eigenen Fehlermeldung abgelehnt
  • die maximal zulässige Anzahl von Zeichen im Titel des Achhievements wurde auf 50 begrenzt
  • Achievements ohne Kategorie werden jetzt im Overview mit einem entsprechenden Hinweis angezeigt
Bugfixes:
  • Problem behoben, bei dem Features ihre Funktion verloren haben
  • Problem behoben, bei dem nur das zuletzt erstellte Achievement im Overview angezeigt wurde
  • Problem mit der Darstellung der Achievement-Bilder behoben
  • Fehler bei der Erkennung von PNG-Formaten behoben

Bugfixes:

  • Problem behoben, bei dem Achievements nicht ohne Bilder erstellt werden konnten
  • Problem bei der Darstellung von mehr als zwei Achievements behoben
